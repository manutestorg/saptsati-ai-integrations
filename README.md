## Getting Started

This is a React + TypeScript + Vite project for the Saptsati marketing website.

### Prerequisites

- Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd saptsati-ai-integrations

# Install dependencies
npm install

# Start the development server
npm run dev
```

The site will be available at `http://localhost:8080`

### Build for Production

```sh
npm run build
npm run preview
```

## Deployment

This project is deployed to GitHub Pages via GitHub Actions. Every push to the `main` branch automatically triggers a build and deployment.

**Live Site**: https://saptsati.com/saptsati-ai-integrations/

## Tech Stack

- **React** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **shadcn/ui** - Component library
- **Framer Motion** - Animations

## Deployment

1. Push to `main` → GitHub Actions builds and deploys to GitHub Pages.
2. Pages source should be set to **GitHub Actions** in repo Settings → Pages.
3. Build artifacts are served from the `dist` folder generated by `npm run build`.

## Custom Domain

The site is published at `saptsati.com/saptsati-ai-integrations/`. Configure DNS CNAME to GitHub Pages if you change domains.
